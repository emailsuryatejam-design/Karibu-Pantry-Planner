## Learning Path

```
Phase 1: Foundations (files 01–04)
  → Core concepts, networking, APIs, databases

Phase 2: Building Blocks (files 05–07)
  → Caching, async communication, distributed systems

Phase 3: Architecture & Design (files 08–09)
  → Patterns, trade-offs

Phase 4: Interview Execution (file 10)
  → Framework, how to structure your answer

Phase 5: Low-Level Design (file 11)
  → OOP, SOLID, design patterns

Phase 6: Practice Problems (files 12–14)
  → Easy → Medium → Hard
```
